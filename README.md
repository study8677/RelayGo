# RelayGo

RelayGo 是一个面向中国大陆用户的 AI API 中转站信息、模型比价与企业服务对比原型。

## 当前能力

- 按站点或模型关键词搜索。
- 按模型、开发票、专票、对公转账、合同、可用状态和认领状态组合筛选。
- 按综合推荐、最低价格、可用率和延迟排序。
- 查看统一口径的模型输入/输出费率、支付、退款和开票资料。
- 选择 2～4 个中转站进行横向比较。
- 匿名提交新站点（原型交互）。
- 响应式桌面与移动端界面。

示例数据均为产品演示数据，不代表真实站点、价格或服务状态。完整产品范围和路线图请阅读 [`docs/PRD.md`](docs/PRD.md)。

## 本地运行

需要 Node.js 20 或更高版本。

```bash
npm install
npm start
```

打开 <http://localhost:4173>。

## 检查

```bash
npm test
npm run check
```
